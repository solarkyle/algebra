1. Variables: Letters That Hold Numbers

Variables are like containers that can hold different values. In algebra, we use letters like x, y, and z to represent numbers that might change.

Think of a variable as a box labeled "x" that can store any number. Once we put a number in, we can use that box in calculations.

def use_variable(x):
    return x + 5  # x can be any number

# Examples
print(use_variable(3))  # Result: 8
print(use_variable(10))  # Result: 15

3. Equations: Finding the Missing Value

An equation is a statement that two expressions are equal. It always includes an equals sign (=).

Solving an equation means finding what value makes the equation true. For example, in x + 5 = 12, the solution is x = 7.

6. Distributive Property: Break It Down

The distributive property lets you multiply a number by each part of a sum separately.

It states that a(b + c) = ab + ac. In words: "multiply what's outside the parentheses by each term inside."

def distribute(a, b, c):
    expanded = a*(b + c)  # The original expression
    distributed = a*b + a*c  # The distributed form
    return f"Original: {expanded}, Distributed: {distributed}"

# Example
print(distribute(3, 4, 5))  # 3(4+5) = 3Ã—4 + 3Ã—5 = 12 + 15 = 27

7. Factoring: Reverse Distribution

Factoring is the opposite of the distributive property. Instead of expanding, we're looking for common factors.

For example, ax + bx can be factored to x(a + b) by pulling out the common factor x.

10. Systems of Equations: Multiple Puzzles at Once

A system of equations is a set of two or more equations with the same variables. We solve them by finding values that make ALL equations true at the same time.

For example, in the system:

x + y = 10
x - y = 4

The solution is x = 7 and y = 3.

def solve_system():
    # Solve: x + y = 10 and x - y = 4
    x = (10 + 4) / 2  # Adding equations: 2x = 14
    y = 10 - x        # Substituting back
    return f"x = {x}, y = {y}"

# Example
print(solve_system())  # x = 7, y = 3
